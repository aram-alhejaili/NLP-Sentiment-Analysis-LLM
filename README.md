# NLP-Sentiment-Analysis-LLM

This project focuses on Sentiment Analysis using both Traditional Machine Learning models and Large Language Models (LLMs).

## Models Used
- Logistic Regression
- SVM
- ALLAM LLM

## Dataset
- IMDB Movie Reviews Dataset

## Project Steps
1. Text Preprocessing
2. Feature Extraction using TF-IDF
3. Model Training
4. Evaluation using:
   - Accuracy
   - Classification Report
   - Confusion Matrix

## Results
The ALLAM LLM achieved the highest accuracy compared to traditional machine learning models.
